# Sales Performance Dashboard

End-to-End Data Analyst project using Python, SQL, and Power BI.

## Project Overview
This project simulates a real-world sales dataset and performs business analysis to generate insights.

## Tools Used
- Python
- Pandas
- SQLite (SQL)
- Matplotlib & Seaborn
- Google Colab

## Key Analysis
• Total Sales by Region  
• Monthly Sales Trends  
• Top Products by Revenue  
• Salesperson Performance  

## Dataset
Generated synthetic dataset with 10,000+ records using Faker.
